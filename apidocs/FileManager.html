PetscErrorCode initializeHDF5(PetscScalar volfrac, PetscScalar timestep, PetscScalar temperature, PetscInt numberElementsX, PetscInt numberElementsY, PetscInt numberElementsZ, PetscScalar penalty, PetscScalar filterRadius, PetscInt numberSteps, PetscInt saveFrequency, PetscInt NoseHooverChainOrder, std::string filePath, bool randomStartingValues, std::string initialConditionsFile, DomainCoordinates domain, std::vector<BoundaryCondition> boundaryConditions)

Creates an HDF5 file for iteration with the provided parameters. This must be called before starting the iteration loop.

Parameters:

  • volfrac – volume fraction setting

  • timestep – timestep setting

  • temperature – system temperature setting

  • numberElementsX – cells in the x direction

  • numberElementsY – cells in the y direction

  • numberElementsZ – cells in the z direction

  • penalty – optimization penalty power setting

  • filterRadius – minimum filter radius setting

  • numberSteps – number of iterations to save

  • saveFrequency – number of iterations for each sample

  • NoseHooverChainOrder – number of particles in the Nose Hoover chain

  • filePath – path to save the HDF5 output file to

  • randomStartingValues – indicates if the positions were set uniformly or not

  • initialConditionsFile – path to pre-set initial conditions if provided

  • domain – domain coordinates of the problem

  • boundaryConditions – groups of boundary conditions for the problem

Returns:

0 or PetscError.

PetscErrorCode saveFinalState(bool saveHamiltionan, HypOptParameters finalState, std::vector<PetscScalar> hamiltonians, std::vector<PetscScalar> compliance, std::vector<PetscScalar> temperatures, std::vector<PetscScalar> LagrangeMultipliers, std::vector<PetscScalar> iterationTimes, std::vector<PetscScalar> timesteps, std::vector<PetscScalar> energyErrors, std::vector<PetscScalar> volFracs, std::vector<PetscInt> solverIterationsSensitivity, std::vector<PetscInt> solverIterationsHamiltonian)

Saves the final state of the system for restarting functionality. Additionally saves metadata from each iteration.

Parameters:

  • saveHamiltionan – option to save the Hamiltonian and Compliance metadata.

  • finalState – final state of the system, including all particle and NH particle positions and velocities.

  • hamiltonians – Hamiltonian at each iteration (if saveHamiltionan option is false this can be NULL).

  • compliance – compliance at each iteration (if saveHamiltionan option is false this can be NULL).

  • temperatures – system temperature at each iteration.

  • LagrangeMultipliers – Lagrangian Multipliers at each iteration.

  • iterationTimes – time to solve each iteration.

  • timesteps – timestep at each iteration. Useful for variable timestep runs.

  • energyErrors – energy error at each iteration.

  • volFracs – volume fraction at each iteration.

  • solverIterationsSensitivity – number of iterations the FEA solver for the sensitivity calculation.

  • solverIterationsHamiltonian – number of iterations the FEA solver for the Hamiltonian calculation.

Returns:

0 or PetscError.

static bool doesFileExist(std::string filePath)

Helper function to determine if a file exists.

Uses the std::ifstream::open function and checks if the fail opened succesfully. This method is imperfect if the file fails to open for another reason, but should work for most cases without getting too complicated.

Parameters:

filePath – the path to check.

Returns:

true if the file exists, false otherwise.

std::string autoAppendFilePath(std::string fileName, std::string fileExtension)

Private helper function which finds a unique file name in the given path.

If the given file alreaqdy exists, “ (1)” is appended to the name. If it still does exists, “ (1)” is incremented until a file name is found which does not exist.
